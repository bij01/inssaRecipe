# project_inssaRecipe
KOSMO TEAM2 1st project (오유현, 김보민, 백인준)

레시피를 검색해주는 PC앱입니다.
ver 1.0
- DB: 메모장 -> JAVA 배열
- GUI: JAVA Swing
- 기능
1) 전체레시피 검색
2) 키워드 검색
3) 랜덤 검색
4) 메뉴버튼 클릭시 화면전환 기능 
(메인 메뉴 화면 전환 시 쓰레드를 활용하여 화면이 위에서 아래로 내려오는 이미지 연출)
5) 레시피 검색시 JFrame사이즈 변경
6) 메인화면 전환 시 랜덤검색 메뉴는 쓰레드로 계속해서 글자 색상이 랜덤으로 변하는 기능
